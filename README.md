# 🍳 Dona Anastásia - Receitas com Afeto
O Dona Anastácia é um espaço digital dedicado a receitas tradicionais, práticas e cheias de nostalgia. Inspirado no sabor das refeições preparadas pelas avós, o projeto une a culinária caseira com a praticidade da web moderna.

# 🚀 Tecnologias e Ferramentas
Para construir este ambiente acolhedor e funcional, foram utilizadas as seguintes ferramentas:
- HTML5 e CSS3: Estrutura e estilização visual do site.
- Figma: Utilizada para o design de interface (UI) e prototipagem.
- Canva: Ferramenta de apoio para criação de identidade visual e elementos gráficos.

# ✨ Funcionalidades 
- 📖 Galeria de Receitas: Organizadas por categorias como doces, salgados e pratos principais.
- 💡 Dicas da Vovó: Truques para fazer o arroz perfeito, bolos fofinhos e carnes suculentas.
- ✍️ Envie sua Receita: Espaço interativo para os usuários compartilharem suas próprias memórias culinárias.
- 📏 Guia de Medidas: Tabela prática para facilitar a conversão de ingredientes.
- 👵 História: Conheça a trajetória da Dona Anastácia.

# 🎨 Identidade Visual
O design foi pensado para ser acolhedor e nostálgico, utilizando tons pastéis e tipografia amigável:
- Cores: Tons de amarelop e laranja (#FFF9BC, #FFFDD3, #FDA036).
- Tipografia: Apricots para títulos e Open Sans para descrições.
